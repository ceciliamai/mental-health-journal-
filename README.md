Mental Health Journal

A simple, browser-based mental health journal app built for the hackathon. Users can log daily moods, write reflections, and save entries locally in their browser — no account required.

About the Project
This app is designed to be a calm, accessible space for mental health check-ins. It lets users track their mood, write private journal entries, and review past reflections, all with a warm, calming design.

Features
- Mood tracking (happy, calm, stressed, sad, excited)
- Free-text journal entries
- Local storage to save entries (no external database needed)
- Clean, mobile-friendly responsive design

How to Run the Project
This is a static HTML/CSS/JavaScript project, so you can run it in 2 simple ways:

Option 1: Open directly in your browser
1. Download the `index.html` file from this repository.
2. Double-click the file to open it in any web browser (Chrome, Firefox, Edge, Safari).
3. Start using the journal immediately!

Option 2: Host with GitHub Pages (Live Demo)
A live demo is available at:  
 https://ceciliamai.github.io/mental-health-journal-/

Theme Alignment
This project aligns with the hackathon theme of mental health and wellness by providing a free, accessible tool for users to reflect on their emotional well-being.
